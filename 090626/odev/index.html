String Split

                            
                                let text = "How are you doing today?";
                                const myArray = text.split(" ");
                                console.log(myArray);
                                
                                result: ["How", "are", "you", "doing", "today?"]
                                
                                split methodu, bir stringi belirli bir ayırıcıya göre parçalara böler ve 
                                bu parçaları bir dizi (array) olarak döndürür.
                                Yukarıdaki örnekte, boşluk karakteri (" ") kullanılarak cümle kelimelere ayrılmıştır.

Array Join

            
                const fruits = ["Banana", "Orange", "Apple", "Mango"];
                const joinedFruits = fruits.join();
                console.log(joinedFruits);
                                        
                result: "Banana,Orange,Apple,Mango"
                                        
                join methodu, bir dizi (array) içindeki elemanları birleştirerek tek bir string oluşturur.
                Varsayılan olarak, elemanlar arasına virgül (",") konur. 
                Ancak, istenirse farklı bir ayırıcı da belirtilebilir.

parseFloat

                
                    parseFloat(10); // 10
                    parseFloat("10"); // 10
                    parseFloat("10.33"); // 10.33
                    parseFloat("10 6"); // 10
                    parseFloat("selam"); // Expected output: NaN

    
                    parseFloat fonksiyonu, bir stringi veya sayıyı ondalık bir sayıya dönüştürür.
                    Eğer string içinde geçerli bir ondalık sayı varsa, bu sayı döndürülür.
                    Eğer string içinde geçerli bir ondalık sayı yoksa, NaN (Not a Number) döndürülür.

setInterval

                    
                        setInterval(myTimer, 1000);
        
                        function myTimer() {
                            const date = new Date();
                            document.getElementById("demo").innerHTML = date.toLocaleTimeString();
                        }

                        setInterval fonksiyonu, belirli bir zaman aralığında (milisaniye cinsinden) bir fonksiyonu
                        tekrar tekrar çağırmak için kullanılır.
                        Yukarıdaki örnekte, myTimer fonksiyonu her 1000 milisaniyede (1 saniye) bir çağrılır 
                        ve geçerli zamanı günceller

clearInterval

                        
                            function stopTimer() {
                                clearInterval(myInterval);
                            }

                            clearInterval fonksiyonu, setInterval ile başlatılan bir zamanlayıcıyı durdurmak için kullanılır.
                            Yukarıdaki örnekte, stopTimer fonksiyonu çağrıldığında myInterval zamanlayıcısı durdurulur ve artık çalışmaz.
